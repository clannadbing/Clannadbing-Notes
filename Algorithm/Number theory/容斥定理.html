丑数 III - 解题思路

1. 容斥定理 - 计算个数

前置知识： [1,i] 中能够被 a 整除的数个数为 i/a 。

为了使用容斥原理，需要首先计算 a 、 b 和 c 的任意两个正整数的最小公倍数，以及三个正整数的最小公倍数。用 gcd(x,y) 表示 x 和 y 的最大公约数，具体做法如下。

  1. 计算 a 和 b 的最小公倍数 lcmAB=a×bgcd(a,b) 、b 和 c 的最小公倍数 lcmBC=b×cgcd(b,c)、a 和 c 的最小公倍数 lcmAC=a×cgcd(a,c)
  2. 计算 a 、b 和 c 的最小公倍数 lcmABC=lcmAB×cgcd(lcmAB,c)

计算上述最小公倍数之后，使用容斥原理计算得到小于等于 x 的丑数个数是 [xa]+[xb]+[xc]−[xlcmAB]−[xlcmBC]−[xlcmAC]+[xlcmABC]

2. 二分

可以将题目转化为：在 [1,i] 中求解能被 a 或 b 或 c 整除的数个数之和大于等于 n 的最小的 i 即是我们的答案。

具备单调性，采用二分法来做，左右边界为 [1,min(a,min(b,c))⋅n]

丑数 III - 实现

class Solution {
public:
    long long lcm(long long x, long long y) {
        return x * y / gcd(x, y);
    } 
    int nthUglyNumber(int n, int a, int b, int c) {
        long long ab = lcm(a, b);
        long long ac = lcm(a, c);
        long long bc = lcm(b, c);
        long long abc = lcm(ab, c);
        int low = 1;
        long long high = min(a, min(b, c))*n;
        while (low <= high) {
            long long mid = low + (high - low) / 2;
            long long cnt = (mid / a) +(mid / b) + (mid / c) - (mid / ab) - (mid / ac) - (mid / bc) + (mid / abc);
            if (cnt < n) {
                low = mid + 1;
            }
            if (cnt > n) {
                high = mid - 1;
            }
            if (cnt == n) {
                high = mid - 1;
            }
        } 
        return low;
    }
};

倍数求和- 解题思路

在 [1,n] 中 m 的倍数有 c=[nm] 个 结合等差数列求和公式，这些数的和为 (1+c)⋅c2⋅m

再结合容斥原理，可以算出 3 或 5 或 7 的倍数之和

class Solution {
public:
    int sum (int k, int n) {
        return ((n / k)*(n / k + 1) / 2) * k;
    }
    int sumOfMultiples(int n) {
        return sum(3, n) + sum (5, n) + sum (7, n) - sum (15, n) - sum (21, n) - sum (35, n) + sum (105, n);
    }
};
